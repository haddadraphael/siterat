# Siterat

Site vitrine statique de Siterat, publie avec GitHub Pages sur :

https://siterat.fr

## Description

Siterat propose la creation rapide de sites web professionnels pour les commerces locaux, independants, restaurants, boutiques, cabinets et petites entreprises.

## Notes

- Le site est en francais par defaut, avec un switch anglais.
- Le formulaire utilise FormSubmit vers contact@siterat.fr.
- La promesse 48h correspond a une premiere version apres reception des contenus necessaires.
